ParrotSMTPAlpha

ParrotSMTP is a tiny SMTP service which accepts and stores any email sent to existing/non-existing recipient within "${parrotsmtp-domain}" domain. It doesn't check sender or recipient address, or anything else.

You can use Burn After Reading client to retrieve stored email. It will be deleted immediately after retrieval.

Core Features

  • Accepts arbitrary messages via SMTP protocol on port 25. Maximum data size accepted for one SMTP connection is limited to 100Kb. Please do not use service for "heavy" messages. It is recommended to use service for short text messages.
  • Number of messages stored in the system is limited. The service deletes old messages automatically. Please retrieve your message shortly after it arrives into the system or it may be deleted by the system before you get it.
  • Read access to messages stored in the system is provided by Burn After Reading client.
